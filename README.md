# Rust

## Cargo

It is the Rust's `build system and package manager`

### Check cargo version.

> cargo --version

### 1.0 Create cargo project.

### 1.1 Initiate cargo. 

> cargo init

### 1.2 Create a new project.

> cargo new `new project name`

### 2.0 Execute cargo.

> cargo run

### 3. Build.

> cargo build
